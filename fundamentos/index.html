EXPLORANDO JAVASCRIPT

A seguinte função tem o objetivo de atualizar o texto de um elemento na página digitando em um input. Se não houver nenhum valor, exibe uma mensagem no console.

A função pega o valor de um input e, se não estiver vazio, cria um novo item de lista (li) com esse valor, adiciona à lista existente e limpa o input; se estiver vazio, exibe uma mensagem no console.
