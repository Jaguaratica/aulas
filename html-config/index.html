EXPLORANDO JAVASCRIPT

Este repositório contém materiais, exercícios e exemplos práticos das aulas de JavaScript.
O objetivo é auxiliar no aprendizado da linguagem, desde os conceitos básicos até tópicos mais avançados.

FUNÇÕES EXEMPLIFICADAS

Teste você mesmo!

function alterarTitulo()

Verifica se o elemento existe
Se existir:
1. Pega o valor do input
2. Define como novo texto do título
Se não existir:
- Mostra "status: null" no console

Input → Verifica → Atualiza título

function descobrirIndice()

Pega texto do input
1. Converte para minúsculo
2. Busca no array de frutas
3. Se encontrado: mostra o índice
Se não: mostra "não encontrado"

Input → Converte → Busca → Mostra resultado

Escreva o nome de uma das seguintes frutas para verificar seu indice:
["Maçã", "Banana", "Laranja", "Uva", "Morango"]

function adicionarLista()

Pega texto do input
Se não vazio:
1. Cria item de lista
2. Adiciona à lista
3. Limpa o campo
Se não existir:
- Mostra "status: null" no console

Input → Valida → Adiciona → Limpa

    function alterarItemLista()

    Pega o valor do input
    1. Verifica se contém vírgula
    2. Separa em índice e novo valor
    3. Converte o índice para número
    4. Verifica se o índice é válido
    5. Altera o item correspondente na lista

    Input → Verifica → Separa → Converte → Valida → Altera

    Digite no formato "índice,novoValor" e clique em "Alterar":
    Exemplo: "2,Óleo" - altera o terceiro item para "Óleo"

    1. Pega valor do item 2. Pega valor do pagamento 3. Se algum valor não for numérico: - Mostra mensagem de erro - Retorna 4. Calcula troco = pagamento - valorItem 5. Se troco for menos que 0: - Mostra "Pagamento insuficiente" Senão: - Mostra "Troco: R$ X.XX"
